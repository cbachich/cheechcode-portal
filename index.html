GeoData Cache

I developed the GeoData Cache website for the Governor's Office of Information Technology. The purpose of the website was to give government agencies in Colorado a single source to share geographic data from the catastrophic flooding in 2013. The application was built on the Ruby on Rails framework, utilizing Leaflet and JQuery for map interaction. Footprint attributes and geometry were stored in a PostgreSQL database using the geospatial PostGIS extension.

Sanborn Online

I designed and developed Sanborn Online, the digital marketplace for Sanborn acquired products. The application was built on the Ruby on Rails framework, utilizing Leaflet and JQuery for map interaction. Footprint GeoJSON was served through GeoServer hooked up to a PostgreSQL database using the geospatial PostGIS extension.

DaBomb Video

DaBomb HTML5 Video Player

A technical demo to show case my understanding of HTML5 and JavaScript. The video player framework was built with AngularJS on the front end and a Ruby on Rails API on the backend. The demo allows a user to control a video and add annotations on the screen during designated timeframes. The annotation text and color is configurable. The user also has the ability to sync annotations from the server, update them, and then sync them back to the server.
