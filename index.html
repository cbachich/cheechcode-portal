GeoData Cache

I developed the GeoData Cache website for the Governorâ€™s Office of Information Technology. The purpose of the website was to give government agencies in Colorado a single source to share geographic data from the catastrophic flooding in 2013. The application was built using open source software to display footprint geometry on an interactive map and provided users with the ability to search, filter, and download data. Footprint attributes and geometry were stored in a PostGIS database.
